## Time option

With this option you can set a timer for the survey to end automatically.

To close the survey in `1 day` and `6 hours` I complete the option like this: `1d6h`

{% hint style="success" %}
Valid formats:

* 1d6h3m2s
* 50h3d1w
* 5h
* 72h2d
{% endhint %}
